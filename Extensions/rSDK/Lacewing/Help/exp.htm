Error string (for on error):
Error$("Lacewing")
This expression gets the description of the last error, if any. Use this when the condition On error is set to TRUE.
Parameters: None.

Lacewing version string:
Version$("Lacewing")
This expression gets the Lacewing client version you are using.
Parameters: None.

Binary to send size:
SendBinarySize("Lacewing")
This expression gets the size of the send binary. You can change the current send binary size with the Binary to send»Resize action.
Parameters: None.

Binary to send address:
SendBinaryAddress("Lacewing")
This expression gets the memory address of the send binary, so you can copy binary data to it.
Parameters: None.

Deny reason (for on [..] denied):
DenyStatus$("Lacewing")
This expression gets the reason your last request was denied. This should be used with the conditions Connect»On connection denied, Name»On name denied, Channel»On channel join denied, or the Channel»On channel leave denied.
Parameters: None.

Get welcome message (for on connect):
WelcomeMessage$("Lacewing")
This expression gets the welcome message the server sends to newly-connected clients. You will not receive a welcome message if your connection is denied.
Parameters: None.

Connection»Host IP address:
HostIP$("Lacewing")
This expression gets the current connected server's IP address (not the name).
Parameters: None.

Connection»Host port:
HostPort("Lacewing")
This expression gets the current connected port.
Parameters: None.

Channel listing»Get name:
ChannelListing_Name$("Lacewing")
This expression gets the current looping channel's name from the channel list (use with the conditions Channel listing»On loop or Channel listing»[With loop name]On loop). To start the loop, use either one of Channel listing»Loop listed channels or Channel listing»Loop listed channels (with loop name) actions.
Parameters: None.

Channel listing»Get peer count:
ChannelListing_PeerCount("Lacewing")
This expression gets the current looping channel's peer count from the channel list (use with the conditions Channel listing»On loop or Channel listing»[With loop name]On loop). To start the loop, use either one of Channel listing»Loop listed channels or Channel listing»Loop listed channels (with loop name) actions.
Parameters: None.

Self»Name:
Self_Name$("Lacewing")
This expression gives you your client's current name. If you have changed your name, you can use the Self_PreviousName$("Lacewing") to get the previous client name. You can change the name by using the Set name action.
Parameters: None.

Self»ID:
Self_ID("Lacewing")
This expression contains your client's current ID. The ID is a 0-based number automatically given to your client by the server, it cannot be changed by the client without reopening the connection.
Parameters: None.

Self»Number of channels:
Self_ChannelCount("Lacewing")
This expression contains a count of all the channels that your client is accessing. You can loop through the channels you are joined to with either the Channel»Select»Loop or the Channel»Loop (with loop name) action.
Parameters: None.

Self»Previous name (for on name changed):
Self_PreviousName$("Lacewing")
This expression contains your client's previous name after you have changed it. You can use the Self_Name$("Lacewing") to get the current client name. You can change the name by using the Set name action.
Parameters: None.

Selected channel»Name:
Channel_Name$("Lacewing")
This expression contains the client's currently selected channel's name. All messages sent to the channel will be sent to this one. If you receive a message on a channel, this expression will be set to the name of that channel. You can select a different channel by using the Channel»Select»By name action.
Parameters: None.

Selected channel»Number of peers:
Channel_PeerCount("Lacewing")
This expression contains the client's currently selected channel's peer count. If you receive a message on a channel, this expression will change to the peer count of that channel. You can select a different channel by using the Channel»Select»By name action.
Parameters: None.

Selected channel»Selected peer»Name:
Peer_Name$("Lacewing")
This expression contains the client's currently selected peer's name. If the peer changes their name, you can use the Peer_PreviousName&36;("Lacewing") expression to get the previous name. If you receive a message on a channel or from a peer, this expression will change to the sending peer's name. You can select a different peer by using the Channel»Peer»Select by name or the Channel»Peer»Select by ID actions.
Parameters: None.

Selected channel»Selected peer»ID:
Peer_ID("Lacewing")
This expression contains the client's currently selected peer's ID. If you receive a message on a channel or from a peer, this expression will change to the sending peer's name. You can select a different peer by using the Channel»Peer»Select by name or the Channel»Peer»Select by ID actions.
Parameters: None.

Selected channel»Selected peer»Previous name (for on peer name changed):
Peer_ID("Lacewing")
This expression contains the client's currently peer's previous name. This should be used with the Channel»Peers»On peer name changed condition.
Parameters: None.
