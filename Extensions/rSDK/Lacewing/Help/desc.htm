Description

The Lacewing Relay Client object is a highly developed communication protocol, enabling you to create online programs, including multiplayer games, chatrooms, file transfer programs, etc.
Due to the advanced functionality of this extension, it is somewhat complicated to understand completely. This help file contains a description of all the parts of the extension - the actions, the conditions, the properties, and the expressions.
You can access more information on the official Lacewing site or search the Lacewing subforum on Clickteam, including:
  •  DizzyDoo's video tutorials
  •  Binary message example
  •  Details on public servers
  •  pylacewing - Lacewing for Python

Legend:
When referencing to other parts of the extension:
Pink: Expression.
Yellow: Parameter.
Cyan: Condition.
Green: Action.
Red: Property.

Quick connection example:
To connect to a server, use the Connect action. Once the Connection»On connect condition is TRUE, use the Set Name action. Once the Name»On name set condition is TRUE, use the Channel»Join action. Once the Channel»On channel joined condition is TRUE, you can send messages to the channel.

Notes:
1. Server messages you send will not go to every user on the server.
2. Channel messages you send will not be sent to yourself.
3. Peer messages will not go to your own client.
4. Peer loops will not include your own client.
5. There is no expression to get your external IP as it is not visible to yourself normally.
