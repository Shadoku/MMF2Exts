Properties

The Lacewing Relay Client has 4 native properties.

Version:
This box displays what version of Lacewing you have. You can also retrieve this using the Version$("Lacewing") expression.

Automatically clear binary:
When you send or blast a binary message, the storage of the send binary is automatically emptied after transmission, rather than you having to use the Clear Stack action afterwards.

Global:
If this checkbox is checked, the object can be access in any frame in the application, including subapplications. If unchecked, only the frame that the object is in can access it.
To identify it from other Lacewing Relay Clients, assuming you have more than one, you will need use the Global Identifier property.

Global Identifier:
Only valid if Global property is being used. This works as an identifier to distinguish between different Lacewing Relay Client objects.
